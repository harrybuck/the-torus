# the-torus-textures

PBR texture pack for [The Torus](https://github.com/harrybuck/the-torus) Obsidian plugin.

The Torus loads textures from `<plugin-dir>/textures/`. This repo distributes that asset bundle as a tarball release so users can opt-in via the plugin's "Install texture pack" button without bloating the main plugin distribution.

Textures sourced from CC0 / public-domain libraries (ambientcg.com, etc).
